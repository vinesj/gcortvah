Решение квадратных уравнений.

Квадратное уравнение — это уравнение вида ax2 + bx + c = 0. Где a - коэффициент перед x2, b коэффициент перед x и с свободный член.

Сушествует много способов решения, но самым универсальным и популярным был и остаётся дискриминант. D = b2 − 4ac. Если дискриминант D меньше 0, то корней нет. Если D = 0, то есть один корень, равный −b/2a. Если D больше 0, то у уравнения две корня .

Теорема Виета ещё один способ решения квадратных уравнений, но она справедлива исключительно для приведённых квадратных уравнений, поэтому для решения уравнений сначало нужно эти уровнения привести к коэфициенту 1 перед x2.
Теорема гласит: если дано x2 + px + q = 0, где x₁ и x₂ являются корнями, то справедливы два равенства: x₁ + x₂ = -p; x₁ * x₂ = q

Франсуа Виет, сеньор де ля Биготьер ( 1540 — 23 февраля 1603) — французский математик, основоположник символической алгебры. По образованию и основной профессии — юрист. Является создатилем теоремы, названной в его честь
К другим научным заслугам: Виет вывел новый тригонометрический метод решения неприводимого кубического уравнения. Виет применил его для решения древней задачи трисекции угла, которую свёл к кубическому уравнению. Так же учёный описал первый пример бесконечного произведения, формула Виета для приближения числа π.

Назад
Теорема Виета.
